# atom-reverser
Reverses your current selections; e.g. false to true etc.

## List of reversible keywords
| Keyword 1 | Keyword 2 |
|-----------|-----------|
| true      | false     |
| 0         | 1         |
| x         | y         |
| width     | height    |
| (         | )         |
| [         | ]         |
| {         | }         |
| +         | -         |
| *         | /         |
| up        | down      |
| left      | right     |
| top       | bottom    |
| hidden    | visible   |
| '         | "         |
