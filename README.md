# atom-reverser
Reverses your current selections; e.g. false to true etc.

## List of reversible keywords

| Keyword 1 | Keyword 2 |
|-----------|-----------|
| and       | nand      |
| or        | nor       |
| false     | true      |
| xnor      | xor       |
